# Daily.py

Daily Use Python Scripts


## Getting Started

* Clone The Repository
* Enter The Directory
* Install Dependencies
* Run The Scripts By The Below Command
```
git clone https://github.com/Chamepp/Daily.py.git
cd Daily.py
pip3 install [dependencyname]
python3 [scriptname].py
```
